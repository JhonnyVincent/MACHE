# Mache

Mache est une marketplace internationale centrée sur Haïti, la Caraïbe et la diaspora.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase
- Architecture prête pour Vercel

## Fonctionnalités
- Homepage premium
- Boutique avec listing
- Fiche produit
- Panier
- Auth pages
- Dashboards buyer / seller / agent / admin
- Vérification publique agent
- Pages légales
- Architecture paiements future
- SQL Supabase complet

## Installation

```bash
npm install
npm run dev
